# **App Name**: Primeiras Mordidas Landing Page

## Core Features:

- Hero Section: Displays the main title, subtitle, app mockup, CTA, and micro-proof to immediately grab the user's attention.
- Features Showcase: Highlights the key features of the app using modern cards with minimalist icons and clear descriptions.
- Target Audience Confirmation: Uses a visual checklist to confirm the app's suitability for specific user needs and concerns.
- Testimonials Section: Showcases real user testimonials in a stories-style format to build trust and credibility.
- Value Proposition: Explains why the app is effective, emphasizing its simplicity, design, practical routine, and organization.
- Pricing and Offer: Presents a clear comparison between the app's price and alternative solutions, highlighting the features included with the app access.
- Guarantee Section: Provides a 7-day guarantee to reassure users and build confidence in their purchase decision.

## Style Guidelines:

- Primary color: Light pink (#FF9EC2) to convey calmness and safety.
- Background color: Soft gray (#F0F0F0) to maintain a clean and minimalist aesthetic.
- Accent color: Slightly darker pink (#FF77B3) for buttons and interactive elements to draw attention without being too harsh.
- Body and headline font: 'Poppins' for a modern, lightweight, and easy-to-read feel. Note: currently only Google Fonts are supported.
- Use linear, minimalist icons in a light pink (#FFB8DA) to match the overall theme.
- Mobile-first design with a clean, intuitive layout optimized for smaller screens. Use ample spacing between sections to enhance readability.
- Smooth scroll animations to create a seamless and engaging user experience.
